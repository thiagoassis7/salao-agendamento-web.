# 💅 Sistema de Agendamento para Salão de Beleza

> 🚧 Projeto em desenvolvimento!  
> Este repositório está sendo construído aos poucos como parte do meu portfólio, com o objetivo de mostrar minha evolução como desenvolvedor em busca de uma vaga de estágio.

Este é um sistema de agendamento online desenvolvido para um salão especializado em cílios e sobrancelhas. O sistema permite que clientes escolham o serviço desejado, selecionem uma data e horário disponíveis, e vejam o valor antes de confirmar o agendamento.

---

## ✨ Funcionalidades atuais

- ✅ Escolha de serviços disponíveis
- ✅ Visualização de preços
- ✅ Seleção de datas e horários disponíveis
- ✅ Bloqueio automático de horários já ocupados
- ✅ Calendário interativo com datas configuráveis (Flatpickr)
- ✅ Interface administrativa para:
  - Gerenciar serviços (nome e valor)
  - Gerenciar horários e datas disponíveis
- ✅ Responsivo para acesso via celular (link na bio do Instagram)

---
